# Java Version Migrations

Recipes for migrating between Java versions in MuleSoft environments.

| Recipe | Description |
|---|---|
| [java8-to-11](./java8-to-11/) | Java 8 to 11: module system, removed APIs |
| [jaxb-removal](./jaxb-removal/) | Replace JAXB with Jakarta XML Binding |
| [java11-to-17-encapsulation](./java11-to-17-encapsulation/) | Strong encapsulation and --add-opens workarounds |
| [powermock-to-mockito](./powermock-to-mockito/) | Replace PowerMock with Mockito inline mocking |
| [cglib-to-bytebuddy](./cglib-to-bytebuddy/) | Replace CGLIB proxies with ByteBuddy |
| [javax-to-jakarta](./javax-to-jakarta/) | javax.* to jakarta.* namespace migration |
